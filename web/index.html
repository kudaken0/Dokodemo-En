どこでも円 Github

調べたい価格をドラッグで選択するだけで日本円に変換された価格を表示します。

オープンソース

どこでも円はオープンソースの拡張機能です。
閲覧情報や個人情報は一切収集しません。

主要硬貨に対応

ドルやユーロなど、海外サイトでよく見る10種類以上の通貨に対応しています。

様々なブラウザサポート

Chromiumベース(Chrome, Edge等)で作られたブラウザで利用できます。

自由にカスタマイズ可能

どこでも円は設定機能が搭載されており、お好みに合わせてカスタマイズできます。

よくある質問

どこでも円は無料でご利用いただけます。
Chromiumベースのブラウザ（Chrome、Edge、Braveなど）で利用できます。
なお、拡張機能のインストールはChromeウェブストアでのみ行うことができます。
